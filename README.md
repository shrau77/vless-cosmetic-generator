# 🦝 VLESS Cosmetic Config Generator

Профессиональный генератор VLESS конфигураций с косметическими настройками для обхода DPI.

## ✨ Возможности

- 🎨 **Косметика для DPI** — фрагментация, noise, MTU
- 📱 **Мульти-клиент** — NekoBox, Husi, Happ, Hiddify, Karing, v2rayNG, Sing-box, Xray
- 🎲 **Рандомизация** — Anti-AI стратегии для обхода ML/DPI
- 📦 **Массовый импорт** — вставляй сотни нод сразу
- 🎯 **Пресеты** — против ТСПУ, для мобильных, максимум

## 🚀 Запуск

### Локально:
\`\`\`bash
bun install
bun run dev
\`\`\`

### GitHub Pages:
Форкни репу → Settings → Pages → Source: GitHub Actions

## 🛠️ Технологии

- Next.js 16
- TypeScript
- Tailwind CSS
- shadcn/ui
